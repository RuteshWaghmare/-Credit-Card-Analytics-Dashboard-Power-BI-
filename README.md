# 💳 Credit Card Analytics Dashboard (Power BI)

A dynamic and interactive Power BI dashboard for analyzing credit card customer behavior, transaction trends, and business performance.

---

## 📊 Description

This project analyzes **57M+ in total revenue** and **667K+ transactions** to uncover insights into credit card customer behavior and transaction trends.  

The dashboard tracks **week-on-week growth (~28%)**, identifies high-value customer segments such as businessmen and white-collar professionals, and highlights key spending patterns across categories like bills and entertainment, enabling data-driven decision-making through interactive visualizations.

---

## 🛠️ Tech Stack

- 📊 **Power BI Desktop** – Main data visualization platform  
- 🔄 **Power Query** – Data transformation and cleaning  
- 📐 **DAX** – Calculated measures and dynamic visuals  
- 🧩 **Data Modeling** – Relationships between tables  
- 📁 **File Format** – .pbix dashboard file  

---

## 📂 Data Source

The dataset consists of structured CSV files containing detailed information about credit card customers and transactions.

It includes:

### 👤 Customer Data
- Client_Num  
- Card_Category  
- Credit_Limit  
- Customer_Acq_Cost  
- Activation_30_Days  
- Delinquent_Acc  

### 💳 Transaction Data
- Total_Trans_Amt  
- Total_Trans_Vol  
- Interest_Earned  
- Exp Type (Expense Category)

### 📅 Time Fields
- Week_Start_Date  
- Week_Num  
- Qtr  
- current_year  

These fields are used for trend analysis and week-on-week performance tracking.

## 📊 Project Overview

### 1. Business Problem
Understanding customer spending behavior and identifying key revenue-driving segments from large transaction data.

### Key Questions
- How do different customer segments contribute to total revenue?  
- Which card category generates the highest revenue?  
- How are transaction patterns changing over time?  
- Which customers show higher credit usage and risk behavior?  

---

### 2. Goal of the Dashboard
To provide a clear and interactive view of revenue trends, customer behavior, and transaction patterns to support better business decisions.

---

### 3. Walkthrough of Key Visuals
- KPI cards showing total revenue (57M+), transactions (667K+), and interest earned (8M+)
- Week-on-week trend analysis for performance tracking  
- Card category breakdown  
- Customer segmentation by job, education, and income  
- Transaction amount and volume analysis  

---

### 4. Business Impact and Insights
- Identifies high-value customer segments contributing most to revenue  
- Highlights top-performing card categories driving business growth  
- Tracks week-on-week revenue trends for performance monitoring  
- Provides insights into spending patterns and credit usage  
- Supports risk management and data-driven decision-making
